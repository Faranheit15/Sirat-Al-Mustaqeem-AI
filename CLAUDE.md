Before starting ANY task:
1. Read docs/CONTEXT.md for project overview, tech stack, and current state
2. Read the README.md inside the specific app you're working on (backend/, frontend/, or mobile/)
3. After completing work, update docs/CHANGELOG.md with what you did
4. If you made an architecture decision, append it to docs/DECISIONS.md

Never modify these instruction files without asking. Never modify docker-compose.yml without asking.
