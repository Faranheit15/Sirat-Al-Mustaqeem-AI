# Sirat Al Mustaqeem AI Frontend

Frontend app placeholder.

The frontend has not been scaffolded yet. The project is currently backend-first: complete and test the FastAPI API through Swagger UI before building the Next.js app.

## Planned Stack

- Next.js 15 with App Router
- TypeScript strict mode
- Tailwind CSS v4
- shadcn/ui
- Supabase Auth
- Bun for package management

## Planned Backend Integration

- Use the FastAPI backend as the only API boundary.
- Pass Supabase access tokens to protected backend routes with `Authorization: Bearer <token>`.
- Use `POST /chat/stream` for SSE chat streaming.
- Use backend conversation endpoints for conversation history.
